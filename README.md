### `EN`
# ᐳ GTAVC-Source code for GTA: Vice City Android

## ᐳ Description
> * **AArch**: `armeabi-v7a`, `arm64-v8a`
> * **Platform**: `Android 15+`
> * **NDK Version**: `NDK26`

## ᐳ Supplement
> * Re-designed render engine components: `RwWorld`, `RpSkin`, `RwCore`, "rphanim", and others
> * Added direct connection to the game
> * Saved in GTA3sfONLY
> * Built-in integration using `JNI_OnLoad`
> * Added path for cache -> `/storage/emulated/0/EdgarVC/`
> * Regular cache [**GTA:VC**](https://pdalife.to/dwn/c0c5d587.html) is suitable
> * TASK: support for online multiplayer game using a plugin

## > CACHE
> * To download the game, download GTA3sfONLY.b in the root folder of the files/
> * And don't forget to add this cache/data/main.txt&main.scm to GameCache/data/
